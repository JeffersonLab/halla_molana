# MOLANA Analysis Repository 

This repository contains the Hall A Moller Polarimetry Analysis Code. The general analysis code is ROOT/C++ based and is used with Bash wrappers. 

## Contents

| Directory | Contents |
|-----------|----------|
| moller_analysis | Contains the general run-by-run analysis for HA Moller polarimetry. |
| moller_details  | Contains CSV files for the sorting information for Moller polarimetry runs. |
| secondary_analysis | Contains the secondary (aka Group) analysis. |
